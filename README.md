# spotify-tracks-analysis
Analysis dashboard for Spotify tracks data from 1930-2023, including visualizations and raw data.
Spotify Tracks Analysis Dashboard

This repository contains a dashboard and dataset for analyzing Spotify tracks from 1930 to 2023. It includes visualizations on track releases, streams, artists (e.g., top artists like Taylor Swift), and audio features (e.g., danceability, energy)

Overview
- **Data Source:** Spotify tracks dataset (cleaned and pivoted in Excel).
- **Key Insights:**
  - Top artists by track count: Taylor Swift (36 tracks), The Weeknd (34), etc.
  - Release trends: Most tracks released in 2022 (401 tracks).
  - Highest streamed songs: e.g., "Blinding Lights" by The Weeknd (3.7B streams).
  - Audio features summary: Average danceability ~69%, energy ~64%.
- Dashboard screenshot:
  ![Spotify Dashboard](spotify_dashboard.png)

## Files
- `Spotify Analysis.xlsx`: Excel file with sheets:
  - "spotify-2023 (3)": Raw track data (e.g., track_name, artist, streams, BPM, etc.).
  - "Cleaned Sheet": Cleaned version of the data.
  - "Pivot Table": Aggregated stats (e.g., track counts by year/artist).
  - "Dashboard": Visual dashboard sheet.
- `spotify_dashboard.png`: Screenshot of the dashboard for quick viewing.

## How to Use
1. Download `Spotify Analysis1.xlsx`.
2. Open in Excel or Google Sheets.
3. Explore the dashboard tab for interactive charts (e.g., yearly growth, top artists like Taylor Swift, monthly releases).
